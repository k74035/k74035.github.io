## List[] / 리스트

파이썬에서 리스트는 여러 개의 값을 순서대로 저장할 수 있는 데이터 구조이다.
리스트는 다양한 데이터 유형 (숫자, 문자열, 객체 등)을 포함할 수 있으며, 
이러한 값을 인덱스를 사용하여 접근하고 조작할 수 있습니다. 
리스트는 파이썬에서 매우 일반적이고 중요한 데이터 구조 중 하나로 반드시 숙지해야한다.

```python
my_list = [1, 2, 3, 4, 5]
print(my_list)
```

**실행결과**
```
[1, 2, 3, 4, 5]
```
